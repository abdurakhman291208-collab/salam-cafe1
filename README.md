# SALAM Restaurant

## Тема проекта
Многостраничный сайт ресторана авторской казахской кухни.

## Описание проекта
Проект представляет собой адаптивный многостраничный веб-сайт ресторана SALAM. 
Сайт разработан с использованием семантической HTML5-разметки и модульной архитектуры CSS.
Основная цель проекта — создать единый визуальный стиль и удобную навигацию для пользователя.

## Страницы сайта
- Главная страница (index.html)
- О ресторане (pages/about.html)
- Меню (pages/menu.html)
- Контакты (pages/contacts.html)

## Используемые технологии
- HTML5 (семантическая разметка)
- CSS3
- Flexbox
- CSS Grid
- Адаптивная вёрстка (mobile / tablet / desktop)
